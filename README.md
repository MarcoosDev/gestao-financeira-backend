# gestao-financeira-backend

API REST para organização financeira pessoal, desenvolvida como projeto de estudos em desenvolvimento backend. Permite registrar perdas, ganhos, planejamento futuro e acompanhar o saldo, com o objetivo futuro de integrar-se a um aplicativo mobile.

>
